YUGO MAFIA

Run stock, money, and permissions from one control centre.

This control system is built for a static Netlify frontend with a Railway-hosted API and PostgreSQL database. Each major tab lives as its own HTML page while sharing the same YUGO MAFIA theme, login flow, and UI system.

Inventory Track quantity, adjustments, reorder thresholds, and stock movements.
Bank Record credits and debits with a running balance and system-linked history.
Access control Create users and decide exactly which pages they can open.
